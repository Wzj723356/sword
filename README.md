# Let's Cultivate immortality

Minecraft修仙主题模组，为游戏添加完整的修仙系统（纹理没做，需要的话加入团队一起，AI生成的，知道一些bug但先搁置一下，拉几个人再搞，理解一下😂）

## 功能特性
- ✅ 完整的修仙等级系统
- ✅ 灵力自动恢复系统
- ✅ 多种武器和功法
- ✅ 灵力驱动的技能系统
- ✅ 丹药系统
- ✅ 阵法系统
- ✅ 修仙HUD界面

## 安装要求
- Minecraft 1.20.1
- Fabric Loader 0.14.21+
- Fabric API 0.92.2+1.20.1

## 核心系统

### 灵力系统
- **灵力上限**：基础100点，随境界提升增加
- **自动恢复**：每2秒恢复5-15点灵力
- **消耗方式**：使用技能、武器攻击

### 功法系统
- **剑法**：基础剑法 → 中级剑法 → 高级剑法 → 剑圣
- **刀法**：基础刀法 → 中级刀法 → 高级刀法 → 刀神
- **枪法**：基础枪法 → 中级枪法 → 高级枪法 → 枪神

### 技能系统
- **火球术**：消耗15点灵力，发射火球
- **冰锥术**：消耗12点灵力，发射冰锥
- **雷电术**：消耗20点灵力，召唤雷电
- **治疗之光**：消耗25点灵力，恢复生命
- **护体盾**：消耗30点灵力，形成护盾
- **瞬移术**：消耗35点灵力，瞬间移动
- **御剑术**：消耗40点灵力，御剑飞行
- **元素爆发**：消耗50点灵力，强大攻击

### 丹药系统
- **聚灵丹**：获得100点修仙经验
- **回血丹**：恢复10点生命和20点灵力
- **筑基丹**：突破到筑基境界
- **灵根重铸丹**：重新随机灵根

## 武器系统
- **修仙剑**：基础伤害3，攻击速度-2.4
- **修仙刀**：基础伤害3，攻击速度-2.2
- **修仙枪**：基础伤害4，攻击速度-2.6
- **飞剑**：远程攻击武器

## 阵法系统
- **修炼阵法**：加速修炼速度
- **攻击阵法**：自动攻击敌人
- **治疗阵法**：恢复生命值
- **传送阵法**：在阵法间传送
